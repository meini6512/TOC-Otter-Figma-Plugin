How to Use This Plugin?

📣 Quick Start

TOC Otter automatically scans your Figma page for frames and sections to create a navigable table of contents.


🏷️ Tagging Frames

To include a frame or section in the TOC, add the prefix # to its name:


  • For example: # Header Section

⚠️ Important

Make sure to include a space after the # symbol.


       ✅ # Hero Section
       ❌ #Hero Section


🤎 Navigation Tips

  • Click any item to jump to that frame
  • Use search to quickly filter items
  • Click Refresh to rescan the page
